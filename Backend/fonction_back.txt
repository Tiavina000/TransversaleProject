1. AUTHENTIFICATION (JWT)
================================================================================

  [POST]   /api/auth/login/
           → Connexion avec username/email/numéro_étudiant + mot de passe
           → Retourne un token JWT (access + refresh) avec les infos utilisateur
           → Vérifie le rôle et l'établissement si fournis
           View: CustomTokenObtainPairView

  [POST]   /api/auth/logout/
           → Déconnexion (côté client : suppression du token JWT)
           View: LogoutView

  [POST]   /api/auth/refresh/
           → Rafraîchir le token JWT access à partir du refresh token
           View: TokenRefreshView (SimpleJWT)

  [GET]    /api/auth/me/
           → Profil de l'utilisateur connecté
           → Retourne : id, username, email, prénom, rôle, établissement,
             niveau (étudiant), spécialité (enseignant), matières enseignées
           View: UserMeView (APIView)

11. SERVICES BACKEND (Moteurs internes)
================================================================================

11.1 NavigationTrie  (core/services/trie_service.py)
──────────────────────────────────────────────────────────────────────────
  → Assistant de Navigation - Algorithme de recherche par préfixe
  → Singleton : l'arbre reste en mémoire après initialisation
  → Indexe les matières, chapitres et leçons
  → Méthodes :
      - initialize_from_db() : Construit l'arbre depuis la BDD
      - insert(mot, metadata) : Insère un mot avec ses métadonnées
      - search_prefix(préfixe) : Recherche DFS, retourne toutes les correspondances

11.2 ScoringService  (core/services/scoring_service.py)
──────────────────────────────────────────────────────────────────────────
  → Moteur de Recommandation - Algorithme de scoring
  → Analyse les performances aux examens
  → Méthodes :
      - analyser_et_recommander(etudiant, matiere) :
          * Récupère les 5 dernières copies de l'étudiant
          * Calcule la moyenne
          * Si moyenne < 10/20 → génère une recommandation
      - _generer_recommandation(...) :
          * Recommande la leçon fondamentale (Chapitre 1, Leçon 1)
          * Score de pertinence = 1.0 - (moyenne / 20.0)

11.3 CurriculumGraph  (core/services/graph_service.py)
──────────────────────────────────────────────────────────────────────────
  → Calcul de Trajectoire - Modélisation par Graphes orientés
  → Construit une liste d'adjacence des leçons dans l'ordre linéaire
  → Méthodes :
      - find_next_step(current_lecon_id) : Prochaine leçon optimale
      - get_full_path(start_lecon_id) : Chemin complet restant

11.4 VoiceCommandProcessor  (core/services/voice_service.py)
──────────────────────────────────────────────────────────────────────────
  → Moteur NLP basique pour commandes vocales
  → Transforme le texte en intention + action
  → Intents reconnus :
      - OUVRIR_LECON : Ouvrir une leçon spécifique
      - PASSER_EXAMEN : Lancer un examen
      - NAVIGATION_GLOBALE : Aller au tableau de bord / profil
      - BOUTIQUE : Ouvrir la boutique
  → Méthodes :
      - process(texte) : Point d'entrée principal
      - _identify_intent(texte) : Correspondance par regex
      - _handle_ouvrir_lecon(texte) : Recherche de leçon par titre
      - _handle_passer_examen(texte) : Recherche d'examen par titre/matière
